🔌 XerefAI (xeref.ai) Integration Notes
Xeref (Second Brain): Built with LangGraph state machine on Next.js 16 App Router — reads/writes Supabase agent_memory table, routes tasks to sub-agents via n8n, reports back to Bugra's dashboard. Model: GPT-4.1 mini.
Creem webhooks: POST /api/webhooks/creem using @creem_io/nextjs — handles all 13 events with idempotency via Supabase webhook_events table.
GPT-4.1 mini: Drop-in replacement for GPT-4o — same OpenAI SDK, change model string to "gpt-4.1-mini". 5× cheaper, 8× larger context window (1M tokens).
Smartlead: Connects via REST API to n8n — trigger sequences directly from Supabase edge functions when new lead rows are inserted.
Agent memory: Store all cross-agent context in Supabase Postgres agent_memory table with RLS — Next.js 16 App Router server components fetch agent state server-side for the dashboard.
